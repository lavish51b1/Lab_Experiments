# Experiment–8: GUI and Backend Connectivity Using Tkinter and SQLite

## Aim
    To design GUI-based applications using Tkinter and connect them with backend databases
    using SQLite.

## Theory
    Tkinter is Python's standard library for building graphical user interfaces (GUIs).
    SQLite is a lightweight, file-based relational database that requires no separate server,
    making it ideal for desktop applications.

        * Tkinter - Provides widgets like Label, Entry, Button, Listbox, Frame, StringVar, etc.
        * SQLite - Used via the sqlite3 module to create, read, update, and delete records.
        * GUI + Backend - Tkinter handles the frontend; SQLite handles persistent data storage.

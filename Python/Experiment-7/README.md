# Experiment–7: File Handling and Exception Handling

## Aim
    To implement file handling operations and apply exception handling techniques in Python.

## Theory
    File handling allows reading and writing data to files for persistent storage.
    Exception handling prevents program crashes by gracefully handling runtime errors.

        * File Handling - open(), read(), write() operations using with statement.
        * Exception Handling - try, except, raise blocks to handle errors.
        * Custom Exceptions - User-defined exception classes by inheriting from Exception.

# Experiment–6: Functions in Python

## Aim
    To study and implement different types of functions in Python including recursive functions, lambda functions and argument types.

## Theory
    Functions are reusable blocks of code that perform a specific task. Python supports various types of functions.

        * User-defined functions - Functions created by the user using the def keyword.
        * Recursive functions - Functions that call themselves with a base case to stop.
        * Lambda functions - Anonymous single-line functions defined using the lambda keyword.
        * Types of arguments - Keyword, Default, and Variable Length (*args, **kwargs).
